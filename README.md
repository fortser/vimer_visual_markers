# ВИМЕР — Визуальные Маркеры Естественной Речи

Инструмент для «очеловечивания» текста: вносит в него реалистичные опечатки, нарушения пунктуации, ошибки пробелов и отклонения в регистре — чтобы машинный текст выглядел написанным живым человеком.

---

## Цель и предназначение

Современные детекторы AI-текста анализируют статистические паттерны: идеальную пунктуацию, правильные пробелы, грамотный регистр. Настоящий человек пишет иначе — он делает опечатки, забывает запятые, ставит лишние пробелы, иногда пишет «е» вместо «ё».

ВИМЕР воспроизводит эти паттерны системно и контролируемо. Каждый тип отклонения реализован как отдельный **маркер** с настраиваемой вероятностью срабатывания. Пользователь сам решает, какие маркеры включать и с какой интенсивностью.

### Профили

Наборы настроек маркеров сохраняются как профили (`profiles/*.json`). Встроенные профили защищены от перезаписи.

| Профиль | Описание |
|---------|----------|
| `default` | Все маркеры на стандартных значениях |
| `light` | Минимальные отклонения |
| `medium` | Умеренные отклонения |
| `aggressive` | Максимальная «человечность» |

Пользовательские профили создаются через GUI («Сохранить как...») или размещением JSON-файла в папку `profiles/`.

### Множитель интенсивности

Глобальный коэффициент (0–300%) масштабирует вероятности всех маркеров одновременно. Позволяет быстро переключаться между лёгким и агрессивным режимом без изменения профиля.

### Seed воспроизводимости

Фиксированный seed гарантирует одинаковый результат при повторной обработке одного текста с теми же настройками. Полезно для тестирования и отладки.

## Запуск

### GUI

```bash
python main.py
```

Горячая клавиша обработки: **Ctrl+Enter**

### CLI

```bash
# Базовая обработка (вывод в stdout)
python cli.py input.txt

# Сохранить результат в файл
python cli.py input.txt -o output.txt

# Указать профиль
python cli.py input.txt -p medium

# Указать профиль, seed и множитель
python cli.py input.txt -p aggressive --seed 42 --multiplier 1.5

# Показать статистику изменений
python cli.py input.txt --stats

# Обработать большой файл (> 5 млн символов)
python cli.py input.txt --force

# Список доступных профилей
python cli.py --list-profiles

# Список всех маркеров с вероятностями по умолчанию
python cli.py --list-markers
```

### Параметры CLI

| Параметр | Описание |
|----------|----------|
| `input` | Входной файл (`.txt`, `.md`). Кодировка определяется автоматически (UTF-8, CP1251, Latin-1) |
| `-o`, `--output` | Путь для сохранения результата. Без этого флага — вывод в stdout |
| `-p`, `--profile` | Имя профиля из папки `profiles/` (без расширения). По умолчанию: `default` |
| `--seed` | Целое число для фиксации случайности |
| `--multiplier` | Число от 0.0 до 3.0. По умолчанию берётся из профиля |
| `--stats` | Вывести статистику изменений в stderr |
| `--force` | Разрешить обработку файлов > 5 млн символов |
| `--list-profiles` | Показать список профилей и выйти |
| `--list-markers` | Показать все маркеры с вероятностями и выйти |
